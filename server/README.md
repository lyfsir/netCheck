## 项目服务模块

 - lyf-auth-service  认证服务
 - lyf-common  公共服务
 - lyf-essearch  检索服务
 - lyf-food  食谱服务
 - lyf-gateway  统一网关服务
 - lyf-third-party-service  第三方服务
 - lyf-topic  话题服务
 - lyf-ums  会员服务

## 技术选型

 - 核心框架：Spring Boot/Spring Cloud
 - 数据库层：mybatis-plus
 - 数据库连接池：Druid
 - 缓存：redis
 - 数据库：mysql5.5以上
